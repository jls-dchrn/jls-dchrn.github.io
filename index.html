My projects

  1. Personal Teacher Assistant🔗

    2. An LLM-Based assistant for childrens needing a tutor to study or do their homework. The goal is for the assistant to help the children by decomposing the problem step by step without giving him the answer. Is also implemented a summary of the current session to assets the children streghts and weeknesses and adapt to his learning pace.

  2. VOC detection🔗

    3. VOCs or Volatile Organic Compounds are a family of indoor gases which can cause different kind of side effects when the go over certain thresholds. This project provide the arduino code and the instructions to build a VOC gas detector.

  3. Generate PDF🔗

    4. Sometime for some benchmarks, we need PDFs which will vary only from a few number of parameters like the number of pages or the number of tables. This code have been used to make different benchmarks on the Docling-serve github. It use the reportlab library. Because there is no tools select the number of pages to fill with text, it have been done experimentally, there is room for improvement.

  4. HGSS pokemon tileset🔗

    5. Some tileset are hard to use because they need to be manually processed. This is the case for the one from SirMalo on DeviantArt. This project provide ready to use tilesets for Pokemon Essential on RPGMaker XP.
